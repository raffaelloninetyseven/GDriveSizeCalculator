Privacy Policy - Drive Folder Size Extension

Last Updated: January 24, 2025
Version: 1.0

Overview

Drive Folder Size Extension ("we," "our," or "the extension") is a Chrome browser extension that calculates and displays the size of folders in Google Drive. This Privacy Policy explains how we collect, use, and protect your information when you use our extension.

Data We Collect

Google Drive Data

  • Folder IDs: Unique identifiers of folders in your Google Drive
  • File metadata: File sizes, folder structure information
  • Authentication tokens: Temporary access tokens to communicate with Google Drive API

Local Storage Data

  • Calculated folder sizes: Cached results to improve performance
  • User preferences: Extension settings and configuration
  • Session data: Temporary data during active use

How We Use Your Data

Primary Purpose

We use your data exclusively to:

  • Calculate and display folder sizes in Google Drive
  • Cache results locally to improve performance
  • Provide the core functionality of the extension

Limited Use Compliance

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

We strictly limit our use of your data to:

  • Providing the folder size calculation service
  • Improving the extension's performance through local caching
  • No other purposes beyond the stated functionality

Data Storage and Security

Local Storage

  • All calculated folder sizes are stored locally in your browser
  • No data is transmitted to external servers (except Google's APIs)
  • Cache data is encrypted and secured within Chrome's storage system

Security Measures

  • All communications with Google APIs use HTTPS encryption
  • Authentication tokens are handled securely and automatically refreshed
  • No permanent storage of sensitive authentication data

Data Sharing and Transfer

We DO NOT:

  • Sell, transfer, or share your data with third parties
  • Use your data for advertising or marketing purposes
  • Transfer data to data brokers or information resellers
  • Use data to determine creditworthiness or for lending purposes

Limited Exceptions:

Data may only be accessed or transferred in these specific circumstances:

  • Legal compliance: When required by applicable law or legal process
  • Security purposes: To investigate abuse or security threats
  • User consent: With your explicit prior consent for specific purposes

Google API Usage

API Scopes

This extension uses the following Google API scope:

  • https://www.googleapis.com/auth/drive.readonly - Read-only access to your Google Drive

Data Minimization

  • We only request the minimum permissions necessary for functionality
  • We only access folder and file metadata required for size calculations
  • We do not access file contents, only metadata

User Rights and Controls

Your Controls

  • Extension Toggle: Enable/disable the extension at any time
  • Cache Management: Clear cached data through the extension popup
  • Account Management: Disconnect your Google account and revoke permissions
  • Uninstall: Remove the extension completely to delete all local data

Data Retention

  • Cached folder sizes are retained until manually cleared or extension is uninstalled
  • Authentication tokens are automatically refreshed and expire regularly
  • No long-term data retention beyond active use

Children's Privacy

This extension is not directed at children under 13 years of age. We do not knowingly collect personal information from children under 13. If you believe a child has provided us with personal information, please contact us immediately.

International Users

This extension may be used globally. By using the extension, you consent to the transfer and processing of your data in accordance with this Privacy Policy and applicable laws.

Changes to This Policy

We may update this Privacy Policy periodically. When we do:

  • We will update the "Last Updated" date at the top
  • Significant changes will be communicated through the extension interface
  • Your continued use constitutes acceptance of the updated policy

Technical Implementation

Data Flow

  1. User authenticates with Google account
  2. Extension requests read-only access to Google Drive
  3. Folder size calculations performed using Google Drive API
  4. Results cached locally for performance
  5. No external data transmission except to Google APIs

Encryption

  • All API communications use TLS/SSL encryption
  • Local storage uses Chrome's secure storage mechanisms
  • No plain-text storage of sensitive information

Limited Use Disclosure

Important: The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements. This extension's use of Google user data is limited to providing and improving the folder size calculation feature as described in this Privacy Policy.

This Privacy Policy is effective as of the date listed above and applies to all users of the Drive Folder Size Extension.
